¿Cómo agregar LKS a tu wallet?

  1. Abre tu billetera (MetaMask, Trust Wallet, etc.) y selecciona la red Polygon.
  2. Si no tienes Polygon configurado, añade esta red manualmente con los siguientes datos: 
    Nombre de red: Polygon Mainnet
RPC URL: https://polygon-rpc.com
Chain ID: 137
Símbolo: MATIC
Explorador: https://polygonscan.com
  3. Luego, haz clic en Agregar Token Personalizado y utiliza esta información: 
    Dirección del contrato: 0x31EF5a8a8Fa124D7270543c1095A45381740c490
Símbolo del token: LKS
Decimales: 18

Detalles del Token

  • Nombre: Lukas Coin
  • Símbolo: LKS
